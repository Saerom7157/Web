행(Row): 테이블의 한 줄, 즉 하나의 레코드를 의미합니다.
SELECT: 데이터를 조회하는 명령어입니다.
FROM: 데이터를 조회할 테이블을 지정하는 명령어입니다.
WHERE: 조건을 지정하여 특정 조건을 만족하는 행만 조회하는 명령어입니다.
AND: 여러 조건을 결합하여 모든 조건을 만족하는 행을 조회합니다.
rowCount 함수: SQL 쿼리를 실행하고, 조건에 맞는 행의 개수를 반환합니다.
